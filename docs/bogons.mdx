The following numbers are considered "bogon" or "martian" space, and will not be permitted on the 48 IX Network.

| Address Block        | Description                     | RFC                                                |
| :------------------- | :------------------------------ | :------------------------------------------------- |

| `100.64.0.0/10`      | Shared Address Space            | [RFC6598](https://tools.ietf.org/html/rfc/RFC6598) |
| `127.0.0.0/8`        | Loopback                        | [RFC1122](https://tools.ietf.org/html/rfc/RFC1122) |
